# Marketplace Copy

## Short description

Curated pastel themes for Visual Studio Code: Leo Dew, Leo Morning, and Leo Velvet in Dark, Dark Deep, and Light variants.

## Long description

Leo is a focused pastel theme collection for Visual Studio Code, named after Leo the dachshund. It ships three distinct theme families, each tuned across dark, deep-dark, and light workspaces.

## Highlights

- 3 curated personalities: Dew, Morning, and Velvet
- 9 total themes: Dark, Dark Deep, and Light for every family
- consistent syntax-role mapping across all variants
- soft UI hierarchy with restrained active accents
- semantic token coverage for modern language servers
- OLED-friendly Dark Deep variants
- bright Latte-inspired Light variants with readable pastel syntax
