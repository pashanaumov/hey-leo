# Changelog

All notable changes to the **Leo** VS Code theme extension.

---

## [1.1.0] - 2026-05-02

### Added
- Added Light variants for Leo Dew, Leo Morning, and Leo Velvet
- Added generated light theme files with readable pastel syntax palettes

### Changed
- Narrowed the published bundle to 3 theme families x 3 variants = 9 themes
- Removed Leo Petal, Leo Burrow, Leo Biscuit, and Leo Dusk from Marketplace contributions
- Updated extension metadata for the new dark/deep/light theme matrix
- Refreshed README positioning around the focused Leo collection

### Themes
- Leo Dew Dark / Dark Deep / Light
- Leo Morning Dark / Dark Deep / Light
- Leo Velvet Dark / Dark Deep / Light

## [1.0.11] - 2026-05-01

### Changed
- Full redesign of all themes — richer syntax palettes, proper UI layer depth
- Renamed extension to **Leo** (after the dachshund)
- Culled to 7 theme personalities × 2 variants (Dark + Dark Deep) = 14 themes
- Applied Catppuccin-style typography (italic functions/types/enums, no bold keywords)
- 3-tier accent system for cursor, activity bar, and scrollbar
- Active tab now has distinct background + accent top border
- Removed all light themes

### Themes
- Leo Velvet Dark / Dark Deep
- Leo Dew Dark / Dark Deep
- Leo Petal Dark / Dark Deep
- Leo Burrow Dark / Dark Deep
- Leo Morning Dark / Dark Deep
- Leo Biscuit Dark / Dark Deep
- Leo Dusk Dark / Dark Deep
